# Security Expert Skill

This skill is designed to identify, analyze, and remediate security vulnerabilities within source code, dependencies, and CI/CD pipelines.

## When to use this skill

- Use this when user asks for a security audit or when you are asked to harden the backend.
- This is helpful for ensuring the code is secure and follows best practices.
- When user asks to generate code related to authentication, authorization, or data encryption.

## How to provide feedback

- Be specific about what needs to change
- Explain why, not just what
- Suggest alternatives when possible
